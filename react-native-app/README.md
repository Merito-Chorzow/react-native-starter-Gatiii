1. Opis Działania

Po uruchomieniu aplikacji wyświetla się lista notatek pobieranych z publicznego API.
Użytkownik może kliknąć dowolną notatkę, aby przejść do ekranu ze szczegółami, gdzie pokazany jest tytuł i treść wybranej notatki.

Z poziomu ekranu głównego dostępny jest też przycisk „Dodaj notatkę”.
Przenosi on do formularza, w którym można wpisać tytuł i opis notatki, a także pobrać swoją aktualną lokalizację GPS.
Po kliknięciu przycisku lokalizacja jest pobierana z urządzenia (po udzieleniu zgody) i wyświetlana na ekranie.

W aplikacji znajduje się także prosty ekran ustawień, na którym wyświetlana jest wersja aplikacji.

2. Funkcje aplikacji

- wyświetlanie listy notatek z API
- podgląd szczegółów pojedynczej notatki
- formularz dodawania nowej notatki
- pobieranie lokalizacji GPS (natywna funkcja urządzenia)
- prosty ekran ustawień z wersją aplikacji

3. Uruchomienie projektu

Wpisz w terminalu "npx expo start"

Aplikacje mozna uruchomic rowniez poprzez expo go w telefonie
